MoneyHana — Project Handoff
Personal finance dashboard, renamed from Buxaplenty. Freemium model, AI insights as the premium feature.
Stack: Next.js 15 App Router, TypeScript, Tailwind v4, shadcn/ui, React Hook Form, Zod v3 (pinned, do not upgrade), Recharts, lucide-react, date-fns, pnpm. Two repos: moneyhana-web and moneyhana-api (FastAPI, Week 3).
Structure: Route groups (auth) and (dashboard). Each page has its own components/ subfolder. Services layer is all mock data until Week 3. Shared types in @/types, shared constants in @/lib/constants.
Design: Flat. border-0 shadow-sm bg-card rounded-sm is the standard card. Primary #1919bc light, #5c5cff dark (not yet added to globals.css). Sidebar always dark. Income green, expenses red/85. Bebas Neue logo only, Plus Jakarta Sans everything else.
Built: Login, Register (split screen, right panel copy not written, tone witty/personal), Dashboard (cumulative balance, period-filtered income/expenses, two charts, recent transactions, insights preview), Transactions (running balance never resets on filter, 30/page, TransactionModal handles add+edit), Goals (colour-coded progress bars, GoalModal, category locked after creation), Insights (grouped by date then type, toggle filter, 30/page), Loader (inline SVG four bars, 25 witty messages, React Compiler strict — no setState in effects).
Conventions: Thin page files, small single-responsibility components (learner on project). Modals: backdrop closes, X top-right, stopPropagation. Edit forms gated by dirty fields. Three-dot DropdownMenu everywhere. Zod uses z.enum not z.string for Category. Amount inputs type=text inputMode=decimal. Running balance always cumulative.
Status: Week 1 complete. Stretch goals Issues 46-49 (Subscriptions UI, Confirmation Flow, Backend, Intelligence). Week 2 next: backend + auth. Still to do: dark mode primary in globals.css, auth panel copy.
